Falling Game

What is this about?

This is a prototype for a small game in JavaScript written for the lecture "Mobile Computing & Internet of Things" at the Karlsruhe Institute for Technology in winter 2019/20.

Rules

The goal is to catch as many falling coins as you can and to avoid the bombs. You start off with 3 lives; each time the basket touches a bomb, you lose a life. There is no time limit. It's game over when you have no more lives left.

Controls

You can control the basket...
  • using the left/right arrow keys.
  • by moving the mouse.
  • using the gyroscope and tilting your device.
Not every device has a gyroscope and not every browser supports it. For example, this game is playable using the gyroscope in Chrome for iOS devices, but not in Safari.
To test whether the gyroscope is available on the browser and device you are currently using, tilt your device to see if the script recognises the device orientation.
